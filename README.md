# Tclock-Win11(仮)

Tclock-Win10からフォークしたプロジェクトです。

■ Windows11 25H2でタスクバーを他のツールで弄ってない環境専用でまだ人柱向けです

## 主な変更点
- Windows11 25H2のスタートメニューの左寄せ、純正時計の非表示化(レジストリ変更で対応)
- 元ソースにあったTClock書式互換エリアを安定動作化調整
- TClock表示エリアの背景色をタスクバーから自動採取する動作の追加(プロパティのWindows11対応)
- iniをUTF-8でも読めるように変更(一部不具合あり)
- ソースをShift-JISからUTF-8に変更
- A APIからW APIに部分移行
